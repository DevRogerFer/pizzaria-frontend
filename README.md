# Sobre o Projeto

Desenvolvimento do frontend de uma aplicação completa para atendimento de uma pizzaria, englobando a criação da interface Web responsiva e do aplicativo Mobile. O projeto foca na construção de interfaces intuitivas, dinâmicas e alinhadas às necessidades do usuário, permitindo o cadastro de categorias e produtos, a visualização dos pedidos no dashboard, a conclusão desses pedidos e a entrega ao cliente. O frontend integra-se diretamente com a API do sistema, garantindo uma experiência fluida, moderna e consistente em todas as plataformas.

# Framework Principal:
  * Next.js 15.5.7 - Framework React com SSR/SSG e App Router:

## Bibliotecas Core:
  * React 19.2.1 - Biblioteca JavaScript para interfaces
  * React DOM 19.2.1 - Renderização do React no navegador

## Linguagem e Ferramentas:
  * TypeScript 5.9.2 - Superset tipado do JavaScript
  * Sass 1.90.0 - Pré-processador CSS

## Comunicação e Estado:
  * Axios 1.11.0 - Cliente HTTP para requisições à API
  * cookies-next 6.1.0 - Gerenciamento de cookies no Next.js
  * js-cookie 3.0.5 - Manipulação de cookies no client-side

## UI/UX:
  * Lucide React 0.544.0 - Biblioteca de ícones
  * Sonner 2.0.7 - Sistema de notificações/toasts

## Infraestrutura de Imagens:
  * Cloudinary - Armazenamento e otimização de imagens (URLs retornadas pela API)
  * Next.js Image Optimization - Otimização automática de imagens

## Autenticação:
  * JWT (JSON Web Tokens) - Tokens armazenados em cookies HTTP-only
  * Middleware do Next.js - Proteção de rotas /dashboard

## Backend/API:
  * API REST em produçãono Railway

## Deploy: 
  * Railway - Plataforma de hospedagem em produção

## Resumo da Stack:
  * Next.js 15 + React 19 + TypeScript + Sass + Axios + Cloudinary + Railway
